## Project Overview

This project was completed as part of the **Udacity AWS Business Intelligence Engineering Nanodegree**, in collaboration with Amazon Web Services. 
It demonstrates an end-to-end, AI-powered automation system built on AWS — designed to solve a real-world inefficiency in career coaching operations.
The fictional company **CarCo** — a career development and coaching organization — serves as the project context. The solution is cloud-native, 
fully scalable, and broadly applicable across any industry where skill-gap analysis, document intelligence, and training recommendation workflows exist.

---

## Problem Statement

Career coaches at CarCo were performing **manual skill-gap analysis** — reviewing student CVs against job descriptions one by one. This created three critical problems:

- **Time-consuming:** Coaches spent excessive time on document review, limiting capacity for meaningful student engagement.
- **Inconsistent:** Results varied by coach, introducing human bias and uneven recommendation quality.
- **Outdated guidance:** Without a structured system, training recommendations lagged behind current job market demands.

This inefficiency directly impacted student career outcomes and eroded trust in the coaching process. A scalable, automated solution was required.
